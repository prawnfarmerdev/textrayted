# textrayted – a text editor in C with raylib

This is a minimal text editor built with Raylib. It's still early but already lets you type, delete, and navigate through text with a cursor.

## What works right now

- Opens a window and renders text line by line
- You can type characters, press Enter for new lines, Tab for spaces
- Backspace deletes with repeat when you hold it down
- Cursor moves as you type and blinks on screen
- Status bar shows line number, column, and character count

## What's next

The plan is to keep adding features like file saving, syntax highlighting, and vim-like modal editing. Check the git history for the original project plan.

## Build and run

You need Raylib installed. Then:

make run

Or compile manually: gcc -Wall -g -std=c11 -o main main.c -lraylib -lm -ldl -lpthread

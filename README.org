#+TITLE: textrayted – a text editor in C with raylib
#+AUTHOR: prawnfarmerdev
#+DATE: 2026-03-15
#+OPTIONS: toc:nil


This is a minimal text editor built with Raylib. It's still early but already lets you type, delete, and navigate through text with a cursor.

** What works right now

- Opens a window titled "Textrayted" and renders text line by line with a monospace font
- You can type printable characters (ASCII 32-125)
- Arrow key navigation: left/right move character by character, up/down move line‑by‑line with column memory
- Press Enter for new lines, Tab inserts 4 spaces (both support key repeat)
- Backspace deletes with repeat when you hold it down (configurable delay/repeat)
- Cursor tracks position in buffer and blinks on screen
- Status bar shows line number, column, and character count
- Efficient text insertion using memmove for proper buffer shifting
- Line-by-line text rendering handles newlines correctly
- Simple buffer implemented as a character array with cursor position tracking

** Technical details

The editor currently uses:
- A fixed-size character buffer (999999 chars max) named `content`
- Cursor struct tracking absolute position, line, and column
- `InsertText()` function that shifts buffer with `memmove` for efficient insertion
- Arrow‑key navigation with line‑aware movement and column memory
- Frame‑based input polling with Raylib’s `GetCharPressed()`, `IsKeyPressed()`, and `IsKeyPressedRepeat()`
- Backspace repeat implemented with timers for initial delay and repeat interval, using `memmove` for proper character deletion
- Text rendering via `MeasureText()` for cursor placement and line‑by‑line `DrawText()`
- Clang‑format configuration for code style (no column limit, custom formatting rules)
- Build system via simple Makefile

** What's next

The plan is to keep adding features like file saving, syntax highlighting, and vim‑like modal editing.

** Build and run

You need Raylib installed. Then:

#+BEGIN_SRC sh
make run
#+END_SRC

Or compile manually:

#+BEGIN_SRC sh
gcc -Wall -g -std=c11 -o main main.c -lraylib -lm -ldl -lpthread
#+END_SRC

** Cross‑compile for Windows (from Linux)

Install mingw‑w64, then run:

#+BEGIN_SRC sh
./build-windows.sh
#+END_SRC

Or manually:

#+BEGIN_SRC sh
x86_64-w64-mingw32-gcc -Wall -O2 -std=c11 -mwindows \
  -Iwinlibs/raylib-5.5_win64_mingw-w64/include \
  -Lwinlibs/raylib-5.5_win64_mingw-w64/lib \
  -static -static-libgcc \
  -o textrayted.exe main.c \
  -lraylib -lwinmm -lgdi32 -lshell32 -lopengl32
#+END_SRC

The resulting `textrayted.exe` is in the current directory. A zip containing the executable is also created.

textrayted - a simple text editor for Windows

This is a minimal text editor built with Raylib.

To run:
1. Extract the zip file
2. Double-click textrayted.exe

Features:
- Arrow key navigation (left/right/up/down)
- Type characters, Enter for new line, Tab for spaces
- Backspace to delete (hold for repeat)
- Status bar shows line number and column

Requirements:
- Windows 10 or later
- May require Visual C++ Redistributable (usually already installed)

Note: This is an early version. Files cannot be saved yet.

Built with mingw-w64 and Raylib 5.5.
